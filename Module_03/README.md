# Module 03: Image Processing Essentials (OpenCV)

## Exercises
1.  **ex01_mat_internals**: Inspect `cv::Mat` memory layout.
2.  **ex02_pixel_access**: Benchmark `at` vs `ptr`.
3.  **ex03_color_spaces**: Implement RGB2Gray.
4.  **ex04_convolution**: Implement 3x3 convolution.
5.  **ex05_morphology**: Implement Dilation/Erosion.
6.  **ex06_histograms**: Compute and Equalize Histograms.
7.  **ex07_thresholding**: Global and Adaptive.
8.  **ex08_gradients**: Sobel derivatives.
9.  **ex09_canny**: Edge detection steps.
10. **ex10_custom_filter**: Separable Gaussian Filter.
