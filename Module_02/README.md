# Module 02: Linear Algebra & Geometry with Eigen

## Exercises
1.  **ex01_matrix_init**: Initialize Eigen matrices (dynamic/fixed).
2.  **ex02_arithmetic**: Add, multiply, and scale matrices.
3.  **ex03_block_ops**: Manipulate ROIs (Regions of Interest).
4.  **ex04_linear_solvers**: Solve Ax=b (LLT, LDLT).
5.  **ex05_eigen_decomp**: Principal Component Analysis (PCA) demo.
6.  **ex06_rotations**: Rotation matrices and Euler angles.
7.  **ex07_quaternions**: SLERP interpolation.
8.  **ex08_se3_transforms**: 4x4 Homogeneous transformations.
9.  **ex09_least_squares**: Fit a line to noisy points.
10. **ex10_geometry_project**: Camera coordinate transformer.
